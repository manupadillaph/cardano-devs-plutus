## **Haskell Validator and Policie Scripts**


### **Cardano Developments Tools** 

*******************************************  

Compilar con:  
> cabal build  

Tambien se pueden instalar para acceder a los ejecutables sin usar cabal.   
> cabal install  
  
Y se crean exe en:  
>  ~/.cabal/bin/deploy-manual-exe  
>  ~/.cabal/bin/deploy-auto-exe
